# Academic Management System – SQL

Relational database system for managing students, courses, enrollments, and grades in an academic institution.

## Technologies
- Microsoft SQL Server
- T-SQL

## Features
- Student and teacher records
- Course and section management
- Enrollment tracking
- Grade reporting
- Views and stored procedures

## Setup
1. Open SQL Server Management Studio
2. Run the schema script
3. Use the provided queries to interact with the data
